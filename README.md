# greywwater-chronicle
  
  An AI-driven persistent world where the consequences of one player's actions reshape the world for everyone who follows.
  
  ## Status
  Week 1 of 8. Building the foundation.
  
  ## What this is
  Most games reset. This one doesn't. greywater-chronicle is an experiment in
  shared persistence: a single world, inhabited by an AI that remembers. When a
  player burns down a village, the next player arrives to ashes. When someone
  brokers peace, the next player inherits the alliance. There is no save slot,
  no rewind — only the chronicle, accumulating.
